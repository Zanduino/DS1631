◆ readTemp()

int16_t DS1631_Class::readTemp ( const uint8_t  device = 0) const

return the current temperature value for a given device number

If no device is specified then device[0] is used. The DS1631 returns raw values in 0.0625°C increments. A check is done to see if there is still an active conversion being processed and will wait until any conversion is complete.

Returns
Temperature in units of 0.0625 degrees Celsius

set the 16 bit value for the temperature alarm on a device and also reset the given alarm

A 0 in "alarmType" equates to low, any other value to high alarm. The alarmTemp is specified in internal units, that is in increments of 0.0625°C

◆ setPrecision()

void DS1631_Class::setPrecision ( const uint8_t  device,
const uint8_t  precision 
) const

Set the precision of the DS1631 to 9, 10, 11 or 12 bits.

This corresponds to increments of 0.5°C, 0.25°C, 0.125°C and 0.0625°C and conversion times of 94ms, 188ms, 375ms and 750ms respectively.

Parameters
[in]deviceDevice number
[in]precision9, 10, 11 or 12 for number of bits precision to use. Invalid values are ignored
